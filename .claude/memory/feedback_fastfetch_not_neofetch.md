ATT never uses or references **neofetch** in any form. If neofetch appears in any suggestion or code, it is always a mistake.

**Why:** ATT exclusively uses `fastfetch` and `fastfetch-git`. Neofetch is a different, unrelated tool and has no place in this project.

**How to apply:** Any mention of `neofetch` in code, logs, comments, or conversation is wrong. Always use `fastfetch` / `fastfetch-git`. If you catch yourself writing neofetch, stop and correct it immediately.
